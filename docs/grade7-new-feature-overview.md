## 二、功能模块详解

### 2.1 基础数据管理 (ms blueprint)

- **年级管理**: 增删改查、排序、启用/停用
- **班级管理**: 增删改查、指定班主任、自动统计学生数
- **科目管理**: 增删改查、满分/及格线设置、排序
- **学期管理**: 增删改查、激活当前学期

### 2.2 学生管理 (class_ blueprint)

- **花名册**: 搜索、筛选、分页查看本班学生
- **增删改查**: 单个添加/编辑/删除（软删除）
- **批量导入**: 下载 Excel 模板 → 填写 → 批量导入
- **数据导出**: 导出本班/全年级学生 Excel
- **学生详情**: 查看学生完整档案（违纪+考勤+成绩+标签）

### 2.3 纪律管理系统

**违纪记录**
- 添加违纪（支持严重程度: 轻微/一般/严重/重大）
- 自动升级机制: 3次轻微→1次一般, 2次一般→1次严重, 2次严重→1次重大
- 自动通知: 班主任 + 家长 实时收到消息
- 缺勤批量转违纪: 从考勤异常一键生成违纪记录
- 批量添加: 一次选择多个学生添加违纪
- 编辑/删除/标记已解决

**违纪统计**
- 多维度统计: 按年级/班级/类型/类别
- 可视化展示趋势

**申诉系统**
- 家长在线提交申诉理由
- 德育处复核（通过/驳回 + 复核意见）
- 自动通知家长复核结果

### 2.4 考勤管理

**每日考勤**
- 快速录入: 出勤/迟到/早退/缺勤/请假
- 迟到/早退阈值可配置
- 支持备注说明

**考勤历史**
- 按日期范围查询历史记录
- 支持导出 Excel

**考勤统计** (attendance_stats blueprint)
- 统计仪表盘（班级排名/出勤率）
- 每日趋势图
- 异常预警（高频缺勤/迟到）
- 学生个人考勤分析

### 2.5 请假审批系统

**家长端**
- 在线提交请假申请（起止日期+原因）
- 查看请假记录和审批状态

**班主任审批**
- 待审批列表
- 通过/拒绝 + 备注
- 自动同步考勤记录

**年级组长审批**
- 二审确认
- 支持批量审批

### 2.6 成绩管理系统

**考试管理**
- 创建考试（月考/期中/期末/模拟）
- 绑定年级

**成绩录入**
- 按考试+班级录入分数
- 支持批量保存

**成绩分析**
- 班级排名 / 年级排名
- 成绩对比（多考试对比趋势）
- 科目分析

**成绩趋势**
- 家长端: 查看孩子成绩走势图

### 2.7 评价系统

**五翼评价 (wings)**
- 五个维度: 德、智、体、美、劳
- 四种评分角色: 教师/家长/自评/互评
- 班级排名
- 勋章系统
- 学生成长档案

**综合素质评价 (quality)**
- 五维指标体系: 道德品质/学业水平/身心健康/艺术素养/社会实践
- 多级指标树（一级→二级）
- 权重可配置
- 多角色评分（自评/互评/教师评/家长评）
- 雷达图可视化
- 评价报告生成

**常规评分 (routine)**
- 班级日常评分（卫生/纪律/两操/礼仪）
- 流动红旗排名
- 按年级/日期统计

### 2.8 任务流转系统

- 德育处创建任务 → 指定目标（年级/班级/个人）
- 自动推送到目标用户
- 目标用户执行 → 提交反馈
- 德育处查看完成情况 → 关闭任务
- 全流程状态跟踪

### 2.9 消息推送系统

**站内消息**
- 发送消息（支持模板）
- 消息中心（收件箱）
- 已读/未读状态
- 支持回复

**实时推送 (SSE)**
- Server-Sent Events 技术
- 新消息实时弹窗提醒
- 违纪/成绩/考勤变动自动推送

**消息模板**
- 6 个系统预设模板
- 自定义模板
- 变量替换: {student_name}, {class_name}, {score}, {date} 等

### 2.10 通知公告

- 德育处/班主任发布通知
- 指定目标（全年级/指定班级）
- 需要签收确认
- 家长端查看 + 签名确认
- 签收统计

### 2.11 家长互动

**期末评语**
- 班主任撰写学期评语（综合评价+优点+待改进+建议）
- 支持批量评语
- 导出 Word/HTML
- 家长端查看

**家长会**
- 创建家长会（时间/地点/目标班级）
- 家长签到（在线+现场）
- 批量签到

**家访记录**
- 记录家访（上门/电话/来校/线上）
- 内容摘要 + 家长反馈 + 后续跟进
- 导出 Excel

**家校沟通**
- 沟通记录
- 提醒家长

### 2.12 活动管理

- 创建活动（标题/描述/类型/时间/地点/参与班级）
- 活动流程: draft → published → ongoing → completed / cancelled
- 学生报名（单个/批量）
- 活动签到（单个/批量）
- 签到统计
- 学生活动记录

### 2.13 学生画像

**问题学生档案**
- 建档（心理/家庭/行为/学习/身体）
- 分等级（红/黄预警）
- 干预措施
- 跟踪记录

**学生标签**
- 自定义标签
- 批量打标签
- 多维度标签组合

**AI 辅助分析**
- 7 条预警规则:
  - 连续缺勤 ≥ 3天 → 🔴 红色预警
  - 本月缺勤 ≥ 5次 → 🟡 黄色预警
  - 本周迟到 ≥ 3次 → 🟡 黄色预警
  - 重大违纪 ≥ 1次 → 🔴 红色预警
  - 严重违纪 ≥ 2次 → 🔴 红色预警
  - 成绩下滑 ≥ 10分 → 🟡 黄色预警
  - 频繁请假 ≥ 3次/月 → 🟡 黄色预警
- 批量查询优化（4 次查询替代原 5446 次）

### 2.14 心理健康

**心理问卷**
- 内置量表（如 MSSMHS-55）
- 在线填写
- 自动计分 + 维度分析
- 问卷统计

**心理健康评估**
- 多种评估方式: 问卷/访谈/观察/家长反馈
- 风险等级: 低/中/高
- 评估结论 + 建议
- 干预方案
- 审核流程

### 2.15 数据看板与报表

**数据大屏**
- 全校关键指标实时展示
- 学生总数/出勤率/违纪统计/班级排名

**报表生成**
- 报表类型: 周报/月报/学期报/自定义
- 自动生成 Excel（含图表: 饼图+柱状图）
- 历史报表管理

**导出汇总**
- 统一导出入口
- 支持: 学生/违纪/常规评分/任务/五翼/家访/评语

### 2.16 系统管理

**账号管理**
- 创建/启停/重置密码
- 角色分配
- 年级/班级绑定

**系统配置**
- 站点名称/学年
- 成绩录入截止日期
- 迟到/早退阈值
- 自动通知开关

**审计日志**
- 所有业务操作留痕
- 按用户名/操作/目标类型/日期筛选

**数据备份**
- 手动创建备份
- 下载/恢复/删除备份

**学期归档**
- 按学期归档数据
- 归档详情查看
- 学期对比
- 恢复归档

### 2.17 小程序 API

- JWT Token 认证 (HS256, 30天有效期)
- 家长端: 首页/成绩/违纪/请假/通知/评语/心理
- 教师端: 首页/学生/违纪/请假审批/考勤统计
- 年级组长: 首页概览
- 消息推送: 违纪/成绩/考勤通知
- 公共: 公告/消息列表

## 四、技术亮点

| 亮点 | 说明 |
|------|------|
| **模块化蓝图架构** | 31 个蓝图独立开发、独立注册、零耦合 |
| **RBAC 权限体系** | 6 角色 + 装饰器 + 数据自动过滤 |
| **双通道认证** | Session (Web) + JWT (小程序)，共享同一用户体系 |
| **SSE 实时推送** | gthread worker + SSE generator，实时消息弹窗 |
| **批量查询优化** | AI 分析从 5446 次查询优化为 4 次，性能提升 77 倍 |
| **自动升级机制** | 违纪次数达标自动升级严重程度并通知 |
| **全流程审计** | AuditLog 记录所有 CRUD 操作 |
| **连接池保护** | pool_pre_ping + pool_recycle 防止断连 |
| **限流防护** | Nginx rate limiting 5r/m 防止暴力破解 |
